
# План: Увеличение видимости фоновых изображений до 96%

## Текущее состояние

Сейчас фоновые изображения в карточках Hero имеют:
- `opacity: 0.15` (15% видимость)
- Градиентный оверлей `from-card/90 via-card/85 to-card/80` (почти полностью перекрывает)

## Изменения

### Файл: `src/components/Hero.tsx`

Для каждой из 3 карточек нужно:

1. **Увеличить opacity изображения** с `0.15` до `0.96` (96%)
2. **Убрать/уменьшить градиентный оверлей** для полной видимости фото
3. **Добавить затемнение только в области текста** для читаемости

| Карточка | Строка | Текущее значение | Новое значение |
|----------|--------|------------------|----------------|
| Выезд за 15 минут | 148 | `opacity: 0.15` | `opacity: 0.96` |
| Сертификаты | 171 | `opacity: 0.15` | `opacity: 0.96` |
| Гарантия | 195 | `opacity: 0.15` | `opacity: 0.96` |

### Решение для читаемости текста

Поскольку фото будет на 96% видимым, нужно обеспечить читаемость текста. Варианты:

**Вариант A**: Добавить полупрозрачный фон только под текстом
```tsx
<div className="relative p-3 md:p-6 lg:p-8 ... bg-card/70 backdrop-blur-sm">
```

**Вариант B**: Заменить градиент на более лёгкий
```tsx
<div className="absolute inset-0 bg-gradient-to-t from-card/60 to-transparent" />
```

Буду использовать **Вариант B** — градиент снизу вверх для читаемости текста.

---

## Итоговая структура карточки

```tsx
<div className="relative overflow-hidden bg-card rounded-xl shadow-sm hover-lift">
  {/* Фоновое изображение на 96% */}
  <div 
    className="absolute inset-0 bg-cover bg-center"
    style={{ backgroundImage: `url('${HERO_CARD_BACKGROUNDS[0]}')`, opacity: 0.96 }}
  />
  {/* Лёгкий градиент снизу для читаемости текста */}
  <div className="absolute inset-0 bg-gradient-to-t from-card/70 via-card/30 to-transparent" />
  
  {/* Контент */}
  <div className="relative p-3 md:p-6 lg:p-8 ...">
    ...
  </div>
</div>
```

## Ожидаемый результат

- Фотографии видны на 96% в каждой карточке
- Текст остаётся читаемым благодаря лёгкому градиенту снизу
- Иконки и заголовки хорошо контрастируют с фоном
