
# План: Синхронизация количества районов Москвы

## Диагностика

Обнаружено расхождение в указании количества районов Москвы на сайте:

| Место | Текущее значение | Правильное значение |
|-------|-----------------|---------------------|
| ServiceAreaMap (карта) | 130 (fallback) | 130 — правильно |
| DistrictPage.tsx (ссылка) | 125 | 130 — нужно исправить |
| Contacts.tsx (ссылка) | 125 | 130 — нужно исправить |
| ServiceDistricts.tsx (кнопка) | 125 | 130 — нужно исправить |
| seoRoutes.ts (комментарий) | 125 | 130 — нужно исправить |

Реальное количество районов в neighborhoods.ts:
- ЦАО: 10, САО: 16, СВАО: 17, ВАО: 16, ЮВАО: 12, ЮАО: 16, ЮЗАО: 12, ЗАО: 13, СЗАО: 8, НАО+ТАО+ЗелАО: 10
- **Итого: 130 районов**

## Безопасность изменений

Изменения затрагивают только текстовые метки — не меняют:
- URL-структуру сайта
- Schema.org разметку
- Маршрутизацию
- Canonical URLs

Индексация не пострадает — это косметические правки.

## План изменений

Заменить жёстко прописанное "125" на динамическое значение.
