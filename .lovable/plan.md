

# Добавить квиз-форму на страницу «Обработка участков»

## Проблема

Страница `ServiceLandingUchastkiPage.tsx` не использует компонент `ServiceQuiz`. Кнопка «Рассчитать стоимость» открывает `CalculatorModal`, но квиз-формы (пошаговый опрос) на странице нет, в отличие от остальных страниц услуг.

## Решение

### Файл: `src/pages/ServiceLandingUchastkiPage.tsx`

1. Импортировать `ServiceQuiz`
2. Определить шаги квиза, специфичные для обработки участков:
   - Что беспокоит? (Клещи / Комары / Борщевик / Несколько вредителей)
   - Площадь участка? (до 6 соток / 6-10 соток / 10-20 соток / более 20 соток)
   - Тип территории? (Дачный участок / Коттеджный посёлок / Парк/сквер / Коммерческая территория)
3. Вставить `<ServiceQuiz>` между секциями «Pricing» и «FAQ» (после строки 157)

Одна точечная вставка, ~20 строк кода.
