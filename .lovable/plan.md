## Выявленные проблемы

### 1. Шаблонные статьи в `pests-articles.ts`
Статьи генерируются по шаблонам с подстановкой данных о вредителях:
```
kak-izbavitsya-ot-{pest}
{pest}-v-kvartire
otkuda-berutsya-{pest}
narodnye-sredstva-ot-{pest}
professionalnaya-obrabotka-ot-{pest}
```

**Проблемы:**
- Одинаковая структура для всех вредителей
- Генерируются программно — видны повторы
- Нет уникального экспертного контента

### 2. Шаблонные статьи в `premises-articles.ts`
Аналогично для объектов (квартира, офис, склад, ресторан):
```
dezinsekciya-{object}
deratizaciya-{object}
podgotovka-k-obrabotke-{object}
stoimost-obrabotki-{object}
```

**Проблемы:**
- Те же паттерны повторов
- Мало уникальных деталей по типам помещений

### 3. ИИ-паттерны, которые нужно удалить
Система уже фильтрует часть паттернов в `TableOfContents.tsx`:
- Emoji в начале строки: ✅❌⚠️📍🔴🟢🟡🎯💡📌
- Фразы: "Важно:", "Следует:", "Рассмотрим подробнее", "Давайте разберём"

**Дополнительные ИИ-паттерны для удаления:**
- "В данной статье мы рассмотрим..."
- "Подводя итоги..."
- "Таким образом..."
- Избыточные перечисления с номерами
- Слишком формальный тон

### Этап 2: Переработка premises-articles.ts (приоритет: высокий)
- Расширить шаблоны с учётом специфики каждого объекта
- Добавить реальные кейсы по типам помещений
- Включить особенности работы (ресторан vs склад)

### Этап 3: Расширение cleanAIContent() (приоритет: средний)
Обновить функцию в `TableOfContents.tsx` для фильтрации дополнительных паттернов:
```typescript
const cleanAIContent = (text: string): string => {
  return text
    // Существующие правила...
    // Новые правила:
    .replace(/^(В данной статье|Подводя итоги|Таким образом)[.,]?\s*/gim, '')
    .replace(/^(Стоит отметить|Как показывает практика)[.,]?\s*/gim, '')
    .replace(/^\d+\.\s*\*\*[^*]+\*\*\s*[—–-]\s*/gm, '') // "1. **Термин** — описание"
};
```

### Этап 4: Проверка статей ID 21+ в blogPosts.ts (приоритет: средний)
Статьи с ID 21 и выше ("Законы", "Кейсы") требуют ручной проверки:

- `shtraf-za-tarakanov-v-kafe`

### Принципы "человечного" текста:
1. **Разговорный тон** — "Вы обнаружили..." вместо "При обнаружении..."
2. **Конкретные цифры** — "за 15 лет мы обработали 12 000 квартир"
3. **Местный колорит** — упоминания районов Москвы
4. **Честность** — "не всегда получается с первого раза"
5. **Истории** — "недавно был случай..." 
6. **Профессиональный сленг** — "оотека", "барьерная защита", "каскадный эффект"
