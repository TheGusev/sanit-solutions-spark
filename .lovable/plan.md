Расширяем существующие 6 ServicePage до уровня ServiceSubpage (добавляем недостающие блоки) и создаем новую посадочную `/uslugi/borba-s-krotami`.

## Шаг 1. Расширить интерфейс ServicePage

**Файл:** `src/data/services.ts`

Добавить опциональные поля в интерфейс `ServicePage`:

- `whenNeeded?: { intro: string; reasons: { icon: string; title: string; description: string }[] }` -- блок "Когда нужна услуга"
- `pricing?: { type: string; area?: string; coldFog: string; hotFog: string; duration: string; highlighted?: boolean }[]` -- таблица цен
- `includedInPrice?: string[]` -- что входит в стоимость
- `guarantees?: string[]` -- гарантии и документы
- `safetyInfo?: string[]` -- безопасность препаратов
- `process?: { step: number; title: string; description: string; details: string[]; duration: string }[]` -- специфичные этапы работы (заменяют общий WorkProcess)
