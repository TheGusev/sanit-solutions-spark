

# План: Изменение роли Владимира Гусева на "Мастер-дезинфектор"

## Текущее состояние

В файле `src/data/blog/types.ts` (строка 39):

```typescript
{
  id: 'gusev-v',
  name: 'Владимир Гусев',
  role: 'Технолог-дезинфектолог',  // ← текущая роль
  experience: '7 лет',
  style: 'technical',
  specialization: ['препараты', 'технологии', 'озонирование', 'туман']
}
```

---

## Изменение

| Файл | Строка | Было | Станет |
|------|--------|------|--------|
| `src/data/blog/types.ts` | 39 | `role: 'Технолог-дезинфектолог'` | `role: 'Мастер-дезинфектор'` |

---

## Результат

После изменения все статьи блога, где автором указан Владимир Гусев (статьи про озонирование, туман, препараты, технологии), будут отображать его должность как **"Мастер-дезинфектор"**.

Это автоматически обновится:
- На странице статьи (author card)
- В Schema.org разметке (jobTitle)
- Во всех местах, где используется `authorRole`
