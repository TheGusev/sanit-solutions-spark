## Фаза 2: Квиз "Рассчитать стоимость"

### Новый компонент: `src/components/ServiceQuiz.tsx`

Многошаговый квиз (3-4 шага) с прогресс-баром:
- Шаг 1-3: Вопросы из `quizSteps` (уникальны для каждой услуги)
- Финальный шаг: Телефон + комментарий + согласие
- Отправка через `handle-lead` с `source: "quiz"` и ответами в `comment`
- Анимированные переходы между шагами
- Кнопки "Назад" / "Далее" (48px+ высота)
- На финальном шаге -- кнопка "Отправить заявку"

### Изменения в `src/data/services.ts`

Добавить в интерфейс `ServicePage`:
```typescript
quizSteps?: {
  question: string;
  options: string[];
}[];
```

Уникальные вопросы для каждой услуги:

| Услуга | Шаг 1 | Шаг 2 | Шаг 3 |
|--------|-------|-------|-------|
| Дезинфекция | Что обрабатываем? (квартира/офис/склад/кафе/медучреждение) | Причина? (болезнь/плесень/ремонт/профилактика) | Нужны документы для Роспотребнадзора? (да/нет/не знаю) |
| Дезинсекция | Какой вредитель? (тараканы/клопы/муравьи/блохи/моль/не знаю) | Тип помещения? (квартира/дом/ресторан/общежитие/склад) | Была ли обработка ранее? (да, не помогло/нет/не знаю) |
| Дератизация | Кого видели? (крысы/мыши/не уверен) | Тип объекта? (квартира/дом/подвал/склад/ресторан) | Где следы? (кухня/подвал/чердак/вентиляция/повсюду) |
| Озонирование | Источник запаха? (табак/пожар/затопление/плесень/другое) | Тип помещения? (квартира/дом/офис/авто) | Как давно запах? (дни/недели/месяцы/годы) |
| Дезодорация | Что пахнет? (труп/моча животных/табак/гарь/сырость/другое) | Тип помещения? (квартира/дом/авто/офис) | Пробовали устранить сами? (да/нет) |
| Демеркуризация | Что разбили? (градусник/лампа/прибор/не знаю) | Где? (квартира/офис/школа/медучреждение) | Когда произошло? (сегодня/вчера/несколько дней назад) |
| Кроты | Площадь участка? (до 6 соток/6-15/15-30/больше 30) | Что повреждено? (газон/грядки/дорожки/фундамент) | Соседи обрабатывали? (да/нет/не знаю) |

## Фаза 3: Тарифы + "Почему проблема возвращается"

### Новый компонент: `src/components/ServiceTariffs.tsx`

3 карточки в ряд (desktop) / стопка (mobile):
- "Эконом" -- базовый набор, цена "от X ₽"
- "Стандарт" -- выделен как "Популярный", расширенный набор
- "Максимум" -- комплексное решение, повторный выезд
- Каждая карточка: название, цена, список включённого (5-6 пунктов), кнопка "Заказать"
- Кнопка "Заказать" открывает звонок (tel:)

### Новый компонент: `src/components/WhyProblemReturns.tsx`

3-4 карточки с иконками, показывается только для dezinsekciya и deratizaciya:
- Каждая карточка: иконка + заголовок + описание причины возврата

### Изменения в `src/data/services.ts`

Добавить в интерфейс:
```typescript
tariffs?: {
  name: string;
  price: string;
  popular?: boolean;
  features: string[];
}[];
returnReasons?: {
  icon: string;
  title: string;
  description: string;
}[];
```

Уникальные тарифы для каждой услуги (примеры):

**Дезинсекция:**
- Эконом (от 1200 ₽): обработка холодным туманом, 1 зона, рекомендации
- Стандарт (от 2500 ₽): + горячий туман, барьерная защита, повторный осмотр
- Максимум (от 4500 ₽): + обработка подъезда, 2 повторных выезда, 3-летняя гарантия

**Дератизация:**
- Эконом (от 1400 ₽): установка приманок, 1 осмотр
- Стандарт (от 3000 ₽): + герметизация ходов, мониторинг 14 дней
- Максимум (от 5000 ₽): + барьерная сетка, ежемесячный контроль 3 мес.

И аналогично для остальных 5 услуг с уникальными формулировками.

**returnReasons для дезинсекции:**
- Тараканы: яйца в вентиляции, соседи не обрабатывали, мусоропровод
- Клопы: яйца в мебели, заражение от соседей, б/у вещи

**returnReasons для дератизации:**
- Незаделанные щели, пищевые отходы, открытый подвал

### Интеграция в `src/pages/ServicePage.tsx`

- `<ServiceTariffs>` после квиза (перед текущей таблицей цен)
- `<WhyProblemReturns>` после тарифов (для нужных услуг)
