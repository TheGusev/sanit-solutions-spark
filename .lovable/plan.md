

# План: Убрать время чтения из блогов

## Проблема

На всех страницах блога после даты публикации показывается время чтения (например "5 мин"), которое нужно удалить.

Текущий вид: `Препараты  26 декабря 2025 г.  5 мин`
Желаемый вид: `Препараты  26 декабря 2025 г.`

---

## Изменение

| Файл | Строки | Действие |
|------|--------|----------|
| `src/pages/BlogPost.tsx` | 139-141 | Удалить блок с readTime |

### Удаляемый код

```tsx
// Строки 139-141 - УДАЛИТЬ:
<span className="text-sm text-muted-foreground">
  {post.readTime}
</span>
```

---

## Результат

После изменения в заголовке статьи блога будет отображаться только:
- Категория (badge)
- Дата публикации

Время чтения больше не будет показываться.
