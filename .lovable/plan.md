

# Оптимизация гео-покрытия кротов: блог + коммерческие лендинги

## Текущее состояние

| Тип | Кол-во | Что покрывает |
|-----|--------|---------------|
| Блог (mole-geo-articles) | 18 | МО города/шоссе (Истра, Красногорск, Барвиха...) |
| NCH-страницы (Tier 2) | 40 | Районы Москвы (Арбат, Тверской, Сокол...) |
| Коммерческий лендинг | 1 | `/uslugi/borba-s-krotami/` (общий) |

## Проблема

1. **NCH для кротов в городских районах Москвы — нерелевантны**. Арбат, Тверской, Басманный — нет дачных участков с кротами. Это тонкий контент, вредит SEO.
2. **Ценные коммерческие запросы МО не перехвачены**: «уничтожение кротов Истра», «борьба с кротами Одинцово», «кроты Красногорск цена» — уходят на блог (информационный), а не на коммерческий лендинг.
3. **Блог vs коммерция**: 18 блог-статей с ценами и CTA фактически каннибализируют коммерческий интент.

## Решение (3 шага)

### 1. Убрать кротов из NCH-генератора для городских районов

В `nchSeeds.ts` — исключить `kroty` из `tier2PestsList`. Кроты не должны генерировать НЧ-страницы для районов Москвы (нет участков).

### 2. Создать коммерческие гео-лендинги для кротов по МО

Добавить в `seoRoutes.ts` список целевых МО-городов для кротов (те же, что в блоге + ещё 5-7 ценных). Маршрут: `/uslugi/borba-s-krotami/[город]/`.

Целевые города (18 + дополнительные):
- Новорижское: Истра, Красногорск, Нахабино, Дедовск
- Рублёвское: Одинцово, Барвиха, Усово, Жуковка
- Дмитровское: Лобня, Долгопрудный, Дмитров, Яхрома, Талдом, Дубна
- Дополнительно: Чехов, Серпухов, Наро-Фоминск, Можайск, Клин, Солнечногорск, Домодедово

### 3. Перепрофилировать блог-статьи на информационный интент

Убрать из 18 гео-блог-статей таблицу цен и прямые CTA. Вместо этого — ссылка на соответствующий коммерческий лендинг. Это решает каннибализацию: блог = информация + перелинковка → лендинг = конверсия.

## Файлы

| Файл | Действие |
|------|----------|
| `src/data/nchSeeds.ts` | Убрать `kroty` из `tier2PestsList`, добавить в отдельный массив `moleCities` |
| `src/lib/seoRoutes.ts` | Добавить массив `moleCityRoutes` с 20-25 городами МО для `/uslugi/borba-s-krotami/[город]/` |
| `src/data/blog/mole-geo-articles.ts` | Убрать `pricingTable` из контента, заменить на ссылку `/uslugi/borba-s-krotami/[город]/` |
| `src/pages/MoleCityPage.tsx` | Создать — коммерческий лендинг «Борьба с кротами в [Город]» (по шаблону district pages) |
| `src/App.tsx` | Добавить маршрут `/uslugi/borba-s-krotami/:city/` → MoleCityPage |
| `vite-plugin-ssg.ts` | Добавить генерацию SSG для moleCityRoutes |

## Структура коммерческого лендинга MoleCityPage

- Hero: «Борьба с кротами в [Город] — от 3 000 ₽»
- Тарифы (Эконом / Стандарт / Максимум из pests.ts)
- Специфика района (тип почвы, ориентиры)
- Квиз (ServiceQuiz с priceMap)
- FAQ (3-4 вопроса, локализованных)
- Перелинковка на соседние города и блог-статью

## Ожидаемый результат

- Минус ~40 тонких NCH-страниц (кроты × районы Москвы)
- Плюс ~22 коммерческих гео-лендинга (кроты × МО города)
- Блог → информационный → перелинковка на лендинг
- Каждый ценный запрос «кроты [город] цена» ведёт на коммерческую страницу
