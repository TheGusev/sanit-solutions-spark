
# Исправление: убрать бред про кротов в квартирах

## Проблема

Файл `src/data/blog/pests-articles.ts` генерирует статьи для грызунов (строка 496-536) по тем же шаблонам, что и для насекомых. Кроты попадают в эту генерацию и получают статьи вроде:

- "Кроты в квартире: признаки и методы борьбы" -- с текстом "через вентиляцию от соседей"
- "Откуда берутся кроты" -- "от соседей через мусоропроводы"
- Таблицы цен "1-комнатная квартира от 1500 руб" для кротов
- FAQ "Нужно ли покидать квартиру на время обработки от кротов?"

Всего 5 таких бредовых статей.

## Решение

**Исключить кротов из генерации `rodentsArticles`** в `src/data/blog/pests-articles.ts`.

Строка 496:
```
// Было:
const rodentPests = pests.filter(p => p.serviceType === 'deratizaciya');

// Станет:
const rodentPests = pests.filter(p => p.serviceType === 'deratizaciya' && p.id !== 'kroty');
```

Кроты уже полностью покрыты 18 гео-статьями из `mole-geo-articles.ts` с правильным контентом про участки, газоны, СНТ и таблицами цен по соткам.

## Что удалится (5 статей)

| Slug | Проблема |
|------|----------|
| `kak-izbavitsya-ot-kroty` | Таблица "1-комн. квартира от 1500 руб" для кротов |
| `v-kvartire-kroty` | "Кроты в квартире", "через вентиляцию от соседей" |
| `otkuda-berutsya-kroty` | "От соседей через мусоропроводы" |
| `narodnye-sredstva-ot-kroty` | Борная кислота от кротов |
| `professionalnaya-obrabotka-ot-kroty` | "Подготовить квартиру к обработке от кротов" |

## Файлы для изменения

1. **`src/data/blog/pests-articles.ts`** -- добавить `&& p.id !== 'kroty'` в фильтр (1 строка)
2. **`vite-plugin-ssg.ts`** -- удалить 5 slug-ов кротовых статей из SSG
3. **`src/lib/seoRoutes.ts`** -- удалить те же 5 slug-ов из маршрутов

## Результат

- Минус 5 бредовых статей с кротами в квартирах
- 18 правильных гео-статей остаются как есть
- Общее количество статей: 186 - 5 = 181
