

# Минимизация Cookie-баннера на мобильной версии

## Проблема

Cookie-баннер занимает слишком много места на мобильных экранах: текст длинный, кнопки крупные, всё растянуто на полную ширину.

## Решение

Файл: `src/components/CookieBanner.tsx`

1. **Уменьшить текст на мобильных** — сократить сообщение, убрать ссылки на "пользовательское соглашение" (оставить только "политику конфиденциальности")  на мобильных через адаптивные классы
2. **Компактные кнопки** — уменьшить размер кнопок, расположить их в одну строку на мобильных
3. **Уменьшить отступы** — `p-2` вместо `p-4` на мобильных
4. **Скрыть иконку Cookie на мобильных** — экономия места
5. **Текст в одну строку** — `text-xs` на мобильных

Итого: баннер станет компактной полоской внизу экрана вместо большого блока.
